# TicTacToe
Курсов проект по Python

# Важно!!!
Изтеглете **Depixel.zip** и отворете zip файла.
Намерете шрифта с името **DePixelHalbfett.ttf** и го изтеглете.


# Как се играе?
Като всеки един морски шах,
Щракнете с мишка върху квадратчетата, за да отбележите X или O.
Ако има победител, ще излезе съобщение на екрана и ще имате възможността да
играете отново с цъкане на бутона 'Play Again'.
Ако няма победител, играта трябва да се рестартира.

**!!!Играта е предназначена за двама играчи!!!**
